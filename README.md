# Turtle Signal Lab

A Turtle Trading Strategy Analyzer built with FastAPI and React.

## Live Demo
- Frontend: https://turtle-signal-lab.vercel.app/
- Backend API: https://turtle-signal-lab.onrender.com/docs

## Tech Stack
- Python, FastAPI, Pandas, NumPy, yfinance
- React, Recharts

## Strategy
Implements the classic Turtle Trading rules using 20-day Donchian channels for entry/exit signals and ATR for position sizing.
